The Challenge

RPC

Require code generation for new data, nodes must constantly adapt.

Shared Memory

Fast but fragile, lock-heavy, and limited to one machine.

Sockets / Pipes

Low-level, verbose, and prone to errors.

Message Queues

Rigid schemas, fixed sizes, and poor scalability.

The Genesis Advantage

No Code Generation

Add new topics or richer data without breakage.

Loose Coupling

Nodes evolve independently. subscribers only take what they need.

Flexible Messaging

Send JSON, structured, or raw data with ease.

Scalable Anywhere

Works seamlessly across threads, processes, and even remote machines.

Key Features

Thread-Per-Class

Each class has its own thread & mailbox for concurrency.

Publisher-Subscriber

Built-in model handles routing and message flow.

One-to-Many / Many-to-One

Broadcast or aggregate data flows effortlessly.

Versatile Data

Handle JSON, structured, or binary data flexibly.

Safe & Reliable

Thread-safe by design with no race conditions.

Powered by ZeroMQ

Built on proven high-performance technology.
